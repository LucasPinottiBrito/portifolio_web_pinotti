# Design System: Portfolio Refactor

## Colors (OKLCH)
- **Background**: `oklch(15% 0.01 260)` (Tinted dark neutral, blueish tint)
- **Foreground**: `oklch(95% 0.005 260)`
- **Primary (Accent)**: `oklch(65% 0.15 260)` (Professional blue)
- **Secondary**: `oklch(40% 0.05 260)`
- **Muted**: `oklch(60% 0.01 260)`
- **Border**: `oklch(25% 0.01 260)`
- **Confidential Badge**: `oklch(70% 0.1 40)` (Amber/Gold tint for attention)

## Typography
- **Headings**: Inter or system-sans, Bold, Tight tracking.
- **Body**: Inter, Regular, 1.6 leading, max-width 75ch.
- **Mono**: JetBrains Mono for technical specs and technologies.

## Scale
- **Base**: 16px
- **H1**: 3.5rem (Fluid: 2.5rem - 4.5rem)
- **H2**: 2.25rem
- **H3**: 1.5rem
- **Body**: 1rem

## Layout
- **Containers**: Max-width 1200px.
- **Spacing**: 8-unit system (8px, 16px, 24px, 32px, 64px, 128px).
- **Rhythm**: Generous vertical white space to let technical content breathe.

## Components
- **Navbar**: Compact, blurred background, subtle border.
- **Hero**: High contrast, bold typography, clear CTAs.
- **Cards**: Minimal borders, no heavy shadows, distinct technical focus.
- **Badges**: Small, mono font, professional colors.
- **Modals**: Full-screen or large dialogs for project depth.
